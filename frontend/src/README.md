# SRC (Kaynak Kod) Klasörü
Bu dizin, React uygulamasının ana kaynak dosyalarını barındırır.
- `main.jsx` : Uygulamanın DOM'a asıldığı giriş noktasıdır.
- `App.jsx` : Sayfaların route (yönlendirme) tanımlarının yapıldığı kapsayıcı dosyadır.
- `index.css` : Global stillerin ve Tailwind tanımlarının yer aldığı stil dosyasıdır.
