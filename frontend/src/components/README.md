# Bileşenler (Components) Klasörü
Uygulama içerisinde birden çok sayfada tekrar tekrar kullanılabilen atomik veya yapısal bileşenler burada bulunur.
- **Layout.jsx**: Navigasyon (Üst Menü) ve Alt Bilgi (Footer) tasarımlarını içeren sarmalayıcı (wrapper) bileşendir. Tüm sayfalar bu layout'un içerisine `<Outlet />` üzerinden yansıtılır.
